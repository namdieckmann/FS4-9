Allgemeine Bedienungshinweise

Für die weiteren Erklärungen wäre es gut, wenn Sie sich mit den Bedienungen von Windows vertraut machen würden. Es steht dafür unter Windows95/98 ein Lernprogramm zur Verfügung. Das ist die einfachste Methode den Umgang mit Windows-Bedienelementen zu erlernen.

Abbildung 1

Für die weiteren Erklärungen haben wir versucht, mit möglichst wenig Text möglichst viel Erklärung zu erreichen. Der Ablauf der Kapitel ist nach der Wichtigkeit und dem logischen Ablauf einer Verwaltung sortiert. Begonnen wird mit der Schüleraufnahme, dann folgt das Eintragen und Verwalten von Fahrstunden und sonstigen Gebühren. Weiterhin folgen Erklärungen zur Zahlungsverwaltung, dann Mahnungen, Auswertungen, Archivierung und Schriftverkehr.

Installation

Die Installationsprogramme sind zwar in englischer Sprache vorhanden, aber sie werden hier Schritt für Schritt erklärt. Für die Installation des Programms sind 2 Schritte notwendig:

Das Deployment Kit

Auf der CD finden Sie im Verzeichnis "DDK" oder "Deployment Kit" eine Datei mit Namen "DDDK32.EXE". Starten Sie bitte diese diese Datei um das sogenannte Deployment-Kit zu installieren. Diese Datei entpackt sich zunächst im "Temp"- Verzeichnis auf Ihrer Festplatte. Anschließend startet das eigentliche Installationsprogramm.

Gehen Sie mit der Taste "Next" weiter bis zum Fenster "Select Components". Hier ist es wichtig folgende Komponenten auszuwählen:

  • 32-Bit Deployment-files
  • 32-Bit ODBC-Drivers (Intersolv)
  • 32-Bit SQL Anywhere 5.0 Runtime Files

Die letzte Option erreichen Sie, wenn Sie den Rollbalken nach unten bewegen.

Gehen Sie solange weiter, bis die Dateien Installiert werden

Die Software FS4

In dem Verzeichnis FS4 auf der CD befindet sich eine Datei "Setup.exe". Starten Sie bitte diese Datei Gehen Sie mit der Taste "Next" weiter, bis Sie zum Fenster "Setup Type". Lassen Sie bitte die Auswahl auf "Typical". Danach geht es weiter mit der Taste "Next" bis zur Installation der Dateien.

Danach ist das Programm FS4 im Startmenü eingetragen, so daß Sie es von dort starten können.

Tastaturbedienung des Programms

Bei Windows ist es möglich Bedienelemente auch mit der Tastatur zu bedienen. Im Hauptmenü sind Buchstaben der Texte mit einem Unterstrich gekennzeichnet. Das ist die zugeordnete Taste für die Bedienung. Hauptmenüeinträge aktivieren Sie mit der "ALT"-Taste und dem angegebenen Buchstaben. Das Menü Schüler aktivieren Sie mit "ALT"-"S". Die dann gezeigten Menüpunkte wählen sie direkt mit einem Buchstaben. Für "Schüler archivieren" benutzen Sie die Taste "A".

Oft benutzte Aktionen sind mit Tastaturkombinationen belegt, die bei der Anwahl der Menüpunkte zu sehen sind. So können Sie z. B. die Aktion "Zahlungen erfassen" mit der Kombination "ALT"+"F5"

In den Fenstern angeordnete Tasten genannt "Buttons", haben teilweise auch einen solchen Unterstrich. Benutzen Sie zur Anwahl die "Alt"-Taste und den entsprechenden Buchstaben.

Teilweise sind auch einfache Bedienungen mit "Return" und "Escape" möglich. Mit "Esc" verlassen Sie ein Fenster, mit "Return" lösen Sie ein Speichern aus.

Die genannten Möglichkeiten können Sie alternativ zur Mausbedienung nutzen, wenn Sie mit der Bedienung des Programms vertraut sind, um eine größere Bearbeitungsgeschwindigkeit zu erreichen.

Menüs + Icons

Wenn kein Bearbeitungsfenster geöffnet ist, sehen Sie am oberen Bildschirmrand das Hauptmenü und darunter eine Icon-Leiste. Dort sind die gebräuchlichsten Aktionen zur Direktanwahl hinterlegt. Wenn Sie mit der Maus auf ein solches Icon zeigen, erscheint nach kurzer Zeit ein erklärender Text. Durch anklicken aktivieren Sie die gewünschte Aktion. Am unteren Bildschirmrand sehen zu jedem gewählten Menüpunkt einen erläuternden Text.

Klicken + Doppelklicken

Die Windows Standardbegriffe sollen hier kurz erläutert werden:

  • Klicken, bedeutet ein kurzes Betätigen der linken Maustaste, wenn sie der Zeiger über der gewünschten Aktion befindet.
  • Doppelklicken, bedeutet ein zweimaliges Betätigen der linken Maustaste in kurzem Abstand.

Schülerdaten

Es sollen hier nur einige Eingabefelder beschrieben werden.

Schülernummer

Es wird standardmäßig eine Schülernummer vorgeschlagen, die sich aus der Jahreszahl (1999), dem Monat (02), dem Tag (20) und einer laufenden zweistelligen Nummer zusammensetzt. Diese Schülernummer ist nur ein Vorschlag und kann verändert werden, wenn Sie ein anderes System der Nummernvergabe verwenden möchten. Beim Verlassen des Feldes wird die Nummer überprüft. Sollte Sie schon vorhanden sein, erhalten Sie eine Meldung. Das Feld können Sie nur mit einer gültigen Nummer verlassen.

Codes

Nach der neuen Verordnung können Sie für einen Schüler zwei Führerscheinklassen eintragen. Die Führerscheinklasse und der Code sind jetzt gleich. Wenn Sie die Klasse über das Auswahlfeld eintragen, wird auch der entsprechende Code eingetragen. Der Code wurde aufgrund der Verarbeitung alter Daten wieder in das Programm übernommen.

 

Alter Preis

Da Sie Ihren Schülern die auf dem Ausbildungsvertrag eingetragenen Preise garantieren müssen, markieren Sie bitte nach Preisänderungen diese Schüler. Bei der Anwahl von Leistungen wird dann der alte Preis eingeblendet. Sie können zusätzlich ein Datum angeben, bis zu dem der alte Preis Gültigkeit haben soll.

Schüler archivieren

Schüler, die ihren Führerschein erworben haben, müssen Sie nicht länger in Ihren aktuellen Daten halten. Mit der Auswahl eines Schülers und dem Betätigen der "Archivieren"-Taste, übertragen Sie alle zum Schüler gehörigen Daten in das Archiv. Das Archiv ist die Filiale mit der Nummer 99. Erreichen können Sie ihr Archiv durch einen Wechsel in die Filiale mit der Nummer 99, das Schülerarchiv. Dort können Sie alle Daten (Fahrstunden, Zahlungen etc.) ansehen und drucken. Der Wechsel der Filiale ist unter "Sonstiges – Adressen + Filialen" beschrieben.

Schülerarchiv

Das Schülerarchiv ist nichts anderes als eine Filiale mit der Nummer 99. Archivieren sollten Sie Fahrschuldaten, um Datenbestände übersichtlich zu halten. Schüler die keine Aktionen mehr haben, verlängern Schüler-, Rechnungs- und Zahlungslisten unnötig. Die internen Datenbanken haben grundsätzlich keine Schwierigkeiten mit großen Datenbeständen. Das Archivieren bringt dem Benutzer übersichtlicheres Arbeiten.

Es können Schüler in das Archiv abgelegt werden, die alle in Rechnung gestellten Leistungen bezahlt haben. Dafür ist im Schülermenü der Menüpunkt "Schüler archivieren" vorhanden.

Datei

Datensicherung

Verwenden Sie viel Sorgfalt auf den Umgang mit Ihren Daten. In die Software integriert ist eine interne Datensicherung. Dabei wird auf dem internen Festplattenlaufwerk im Verzeichnis "C:\aaa\fs4\daten\backup" eine Kopie der aktuellen Datenbanken erstellt. Es werden dort zwei Dateien abgelegt:

  • Fs4.db
  • Fs4.log

Da es unterschiedliche Systeme für Datensicherung auf externen Datenträgern gibt, sollten Sie sich für eines dieser Systeme entscheiden und dafür sorgen, daß die oben genannten Dateien auf diesen Datenträger kopiert werden.

Bedenken Sie bitte, das sich im laufe der Zeit eine große Datenmenge ihrer Fahrschule im Rechner entsteht. Nur eine Datensicherung auf einem externen Datenträger bringt Ihnen Datensicherheit bei einem Rechnerausfall.

Technische Daten

Datenbank

Für die Daten wird eine SQL-Datenbank SQL-Anywhere von Sybase verwendet. Als Besonderheit kann diese Datenbank die von Ihnen erzeugten Abfragen selbst optimieren. Einmal getätigte Abfragen laufen damit beim zweiten Verwenden schneller.

Programm

Für das Programm wird ebenfalls eine Technologie von Sybase verwendet. Die mit Powerbuilder erstellte Software wird mit Hilfe des implementierten Watcom C-Compilers in Programmcode gewandelt.
